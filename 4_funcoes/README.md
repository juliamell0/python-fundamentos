# 🔧 Funções em Python

Este diretório contém meus estudos sobre funções na linguagem Python.

Funções permitem organizar o código em blocos reutilizáveis, tornando o programa mais limpo, modular e eficiente.

---

## 📚 Conteúdos Estudados

- Definição de funções com `def`
- Parâmetros e argumentos
- Retorno com `return`
- Funções com múltiplos parâmetros
- Escopo de variáveis
- Funções aninhadas
- Funções com valores padrão

---

## 🧠 Objetivo

- Organizar melhor o código
- Evitar repetição
- Melhorar a legibilidade
- Aplicar modularização
