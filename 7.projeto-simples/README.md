# 🗂 Projeto Simples CRUD em Python

Este projeto tem como objetivo aplicar conceitos fundamentais da programação através da implementação de um sistema CRUD.

CRUD significa:

- Create (Criar)
- Read (Ler)
- Update (Atualizar)
- Delete (Deletar)

---

## 📚 Funcionalidades Implementadas

- Cadastro de registros
- Listagem de dados
- Atualização de informações
- Remoção de registros
- Menu interativo
- Validação de entradas

---

## 🧠 Conceitos Aplicados

- Estruturas condicionais
- Estruturas de repetição
- Listas
- Funções
- Organização lógica do código

---

## 🎯 Objetivo

- Consolidar fundamentos da programação
- Simular sistema real simples
- Praticar lógica e organização
